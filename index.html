CSS Adlandırma Yöntemleri

Web projelerinde stil dosyalarını organize etmek için popüler adlandırma yöntemlerini öğrenin.

1. BEM (Block, Element, Modifier)

BEM, CSS sınıflarını blok, eleman ve modifikasyon olarak böler.

.header {
  background-color: #333;
  color: white;
  padding: 20px;
}
.header__logo {
  width: 100px;
  height: 50px;
  background-color: white;
}
.header__logo--small {
  width: 50px;
  height: 25px;
}

Karmaşık Senaryo:

<div class="product-card">
  <h2 class="product-card__title">Ürün Adı</h2>
  <div class="product-card__image product-card__image--featured"></div>
  <p class="product-card__description">Ürün açıklaması...</p>
  <button class="product-card__button product-card__button--buy">Satın Al</button>
</div>

2. SMACSS (Scalable and Modular Architecture for CSS)

SMACSS, CSS kurallarını kategoriye ayırarak projeyi düzenli hale getirir. 

/* Taban */
body {
  font-family: Arial, sans-serif;
}

/* Düzen */
.layout-header {
  background-color: #333;
  color: white;
  padding: 20px;
}

Karmaşık Senaryo:

<body class="theme-light">
  <header class="layout-header">
    <h1 class="layout-header__title">Blog Başlığı</h1>
  </header>
  <main class="layout-main">
    <article class="post">
      <h2 class="post__title">Yazı Başlığı</h2>
      <p class="post__content">Yazı içeriği...</p>
      <button class="button is-active">Devamını Oku</button>
    </article>
  </main>
</body>

3. OOCSS (Object-Oriented CSS)

OOCSS, taşınabilir ve tekrar kullanılabilir nesneler oluşturmayı hedefler. 

.box {
  padding: 20px;
  border: 1px solid #ccc;
}

.box--rounded {
  border-radius: 10px;
}

Karmaşık Senaryo:

<div class="gallery">
  <div class="box box--rounded">
    <img src="image1.jpg" alt="">
  </div>
  <div class="box">
    <img src="image2.jpg" alt="">
  </div>
</div>

4. Atomik CSS

Atomik CSS, stil özelliklerini küçük ve birleşimiş formlar olarak uygular. 

.mt-2 {
  margin-top: 20px;
}

.p-3 {
  padding: 30px;
}

.bg-blue {
  background-color: blue;
}

.text-white {
  color: white;
}

Karmaşık Senaryo:

<div class="p-3 bg-blue text-white mt-2">
  <h2 class="text-lg">Kullanıcı Adı</h2>
  <p>Bu bir profil açıklamasıdır.</p>
</div>

Özet:

Bu yöntemler, CSS sınıflarını organize etmek ve stil çatışmalarını azaltmak için kullanılır.

  • BEM: Modüler yaklaşım.

  • SMACSS: Kategorilere ayrım.

  • OOCSS: Taşınabilir nesneler.

  • Atomik CSS: Stilözgülü şablonlar.z

Bu yöntemlerden birini seçerek projelerinizi daha düzenli ve sürdürülebilir hale getirebilirsiniz.
